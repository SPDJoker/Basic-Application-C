## MacOS Installation

### Requirements

* [Git](https://git-scm.com/downloads)
* [Homebrew](https://brew.sh/)

### Set up

1. Copy the repository using:
```Makefile
$ git clone https://github.com/SPDJoker/Basic-Application-C.git
```

2. Download the dependencies locally:
```Makefile
$ brew install glfw
```
